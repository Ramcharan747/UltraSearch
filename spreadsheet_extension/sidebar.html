How to use UltraSearch

1. Select Query Mode: Choose Active Selection or Manual Input.

2. Reference Columns: Reference sheet columns in your prompt by typing / or using curly braces, like {{Company}}.

3. Define Output: Select which column of your sheet to write the extracted results to.

4. Click Run: The AI will fetch the web overview for each row and write the exact information requested.
